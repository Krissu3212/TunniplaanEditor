Tunniplaan editor (alustatud 25.12.2020, lisatud Githubi hiljem)

runEditor.py failis muuta IMAGENAME variable-t vastavalt töödelda tahetud faili nimele.

Programm lisab tunniplaani tulpadele suuremad vahed, muudab tausta tumedaks (saab seadistada lihtsasti ka muuks värviks koodis) ning lisab tundide ajad.
